- 파이썬에서 일반 문자열을 처리할 때는 각 유니코드 문자를 저장하는 방법에 대해 걱정하지 않아도 됨.

- 외부데이터를 가져올 때는 저장방법에 대해 알아야 하는데
    1. 문자열을 바이트로 encoding : 인코딩, 무언가를 코드로 바꾼다는 뜻
    2. 바이트를 문자열로 decoding : 디코디

- 바이트당 문자의 종류
    1. 1바이트 : Ascii
    2. 2byte : Latin
    3. 3byte : ?
    4. 4byte : asia , 기호

- 문자열에 따른 바이트 차이

    1. cafe : e는 Latin
